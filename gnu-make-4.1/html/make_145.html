15. Makefile Conventions

This describes conventions for writing the Makefiles for GNU programs. Using Automake will help you write a Makefile that follows these conventions. For more information on portable Makefiles, see POSIX and Portable Make Programming: (autoconf)Portable Make section ‘Portable Make’ in Autoconf.
