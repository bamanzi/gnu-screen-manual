20.11 Time

Command: time [string]

(C-a t, C-a C-t)
Uses the message line to display the time of day, the host name, and the load averages over 1, 5, and 15 minutes (if this is available on your system). For window-specific information use info (see section Info). If a string is specified, it changes the format of the time report like it is described in the string escapes chapter (see section String Escapes). Screen uses a default of ‘%c:%s %M %d %H%? %l%?’.
