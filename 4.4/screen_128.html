20. Miscellaneous commands

The commands described here do not fit well under any of the other categories.

20.1 At   Execute a command at other displays or windows.
20.2 Break   Send a break signal to the window.
20.3 Bumpleft   Swaps window with previous one on window list.
20.4 Bumpright   Swaps window with next one on window list.
20.5 Collapse   Collapses window list.
20.6 Debug   Suppress/allow debugging output.
20.7 License   Display the disclaimer page.
20.8 Nethack   Use nethack-like error messages.
20.9 Nonblock   Disable flow-control to a display.
20.10 Number   Change the current window’s number.
20.11 Time   Display the time and load average.
20.12 Verbose   Display window creation commands.
20.13 Version   Display the version of screen.
20.14 Zombie   Keep dead windows.
20.15 Printcmd   Set command for VT100 printer port emulation.
20.16 Rendition   Change text attributes in caption for flagged windows.
20.17 Sorendition   Change the text highlighting method.
20.18 Attrcolor   Map attributes to colors.
20.19 Setsid   Change process group management.
20.20 Eval   Parse and execute arguments.
20.21 Maxwin   Set the maximum window number.
20.22 Backtick   Program a command for a backtick string escape.
20.23 Screen Saver   Define a screen safer.
20.24 Zmodem   Define how screen treats zmodem requests.
10.9 Mousetrack   Set whether screen should track mouse events.
