9. Regions

Screen has the ability to display more than one window on the user’s display. This is done by splitting the screen in regions, which can contain different windows.

9.1 Split   Split a region into two
9.2 Focus   Change to the next region
9.3 Only   Delete all other regions
9.4 Remove   Delete the current region
9.5 Resize   Grow or shrink a region
9.6 Caption   Control the window’s caption
9.7 Fit   Resize a window to fit the region
9.8 Focusminsize   Force a minimum size on a current region
9.9 Layout   Manage groups of regions
