10. Window Settings

These commands control the way screen treats individual windows in a session. See section Virtual Terminal, for commands to control the terminal emulation itself.

10.1 Naming Windows (Titles)   Control the name of the window
10.2 Console   See the host’s console messages
10.3 Kill   Destroy an unwanted window
10.4 Login   Control ‘/etc/utmp’ logging
10.5 Mode   Control the file mode of the pty
10.6 Monitoring   Watch for activity or inactivity in a window
10.7 Windows   List the active windows
10.8 Hardstatus   Set a window’s hardstatus line
